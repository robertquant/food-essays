## 创作要求

### 基本要求
- **字数**：3000-6000字
- **视角**：80后第一人称
- **风格**：散文随笔，感情真挚，细节丰富
- **素材使用**：每篇至少引用知识库中3个素材

### 时间线参考
```
1980s初 → 票证时代尾声
1980s中 → 进口零食萌芽
1987    → 肯德基进入中国
1990    → 麦当劳进入中国
1990s   → 方便面时代
2000s   → 菜系大融合
2010s   → 外卖时代
2020s   → 怀旧经济
```

### 主题方向
1. **童年记忆**：零食、第一次吃肯德基/ McDonald's
2. **时代见证**：从票证到外卖的变迁
3. **地方记忆**：福安美食、家乡味道
4. **情感记忆**：学生时代、职场记忆

## 创作流程

1. **选题**：从题材索引中选择未创作的题材
2. **查重**：检查知识库确保素材可用
3. **创作**：使用素材库完成散文
4. **保存**：保存到 essays/ 目录
5. **更新索引**：在题材索引中标记已用
6. **提交**：Git commit & push
